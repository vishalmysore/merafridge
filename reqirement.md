Grocery Space Preview — Plain English Requirements
Goal

A user should be able to check whether groceries will physically fit inside their fridge before buying them.

Core Requirements
1. Scan the Fridge Once

The app should allow the user to scan their fridge using their phone camera.

The scan should understand:

fridge height
fridge width
fridge depth
shelf locations
door compartments
freezer section

The scan should be saved so the user does not need to rescan every time.

2. Create a Virtual Fridge

After scanning, the app should create a virtual version of the fridge.

The virtual fridge should:

match real dimensions
contain shelves
remember spacing
behave like the real fridge layout
3. Add Grocery Items

The user should be able to add groceries they plan to buy.

Ways to add:

type item manually
select common grocery items
scan barcode
use camera recognition

Each item should have approximate size.

Example:

milk carton
egg box
yogurt tub
vegetables
soda bottle
4. Simulate Placement

The app should try placing groceries inside the fridge automatically.

It should determine:

where items fit
which shelf works best
whether rearrangement is needed
if something does not fit
5. Show Fit Result

The app should clearly tell the user:

everything fits
some items fit
nothing fits
rearrangement required

Simple visual feedback:

green = fits
yellow = tight fit
red = impossible
6. Show Remaining Space

After simulation, the app should display:

how much space remains
which shelves are full
unused fridge areas
occupancy percentage
7. Allow Rearrangement

The user should be able to move items around manually.

Examples:

drag milk to another shelf
rotate item sideways
stack containers
8. AR / WebXR View

The user should be able to view the fridge in AR.

They should see:

fridge overlay in real space
groceries placed virtually
shelf usage visually
9. Remember Current Fridge State

The app should remember what is already inside the fridge.

This allows future predictions to be realistic.

10. Shopping Mode

While shopping, the user should be able to:

add planned items quickly
instantly see if they fit
adjust cart decisions before checkout
Nice-to-Have Features
suggest best shelf placement
suggest removing old items
recommend reorganization
identify wasted space
show “tight packing” mode
estimate clutter level
Important UX Requirement

The app should feel like:

“Point → Add groceries → Instantly know if it fits”

No complicated setup.

Fast and visual.

Success Condition

A user should answer this question in under 30 seconds:

“Can I buy this and still fit it in my fridge?”
